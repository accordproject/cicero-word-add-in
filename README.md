# Cicero Word Add-in

[![accord project](https://img.shields.io/badge/powered%20by-accord%20project-19C6C8.svg)](https://www.accordproject.org/)

Microsoft Word Add-in for editing texts of Smart Legal Contracts.

## For Developers

Please consult the [DEVELOPERS][addindevelopers] file for the instructions to set up the project on your development environment and visit the [CONTRIBUTING][addincontributing] file to know about Accord Project contributing guide.
